Description du projet

Cette application de gestion des clients a été développée avec Nest.js pour répondre aux besoins d'une petite entreprise souhaitant gérer sa clientèle de manière simple et efficace.

L'application propose une interface web intuitive pour la gestion des clients ainsi qu'une API REST complète pour l'intégration avec d'autres systèmes.

Fonctionnalités

L'application offre les fonctionnalités suivantes :

  • Ajout de nouveaux clients avec nom, email et téléphone optionnel
  • Consultation de la liste complète des clients
  • Modification des informations d'un client existant
  • Suppression d'un client
  • Interface web moderne et responsive
  • API REST pour l'intégration avec d'autres applications

Technologies utilisées

Nest.js

Framework Node.js pour le développement d'applications server-side évolutives.

TypeScript

Langage de programmation typé basé sur JavaScript.

PostgreSQL

Base de données relationnelle pour la persistance des données.

TypeORM

ORM pour TypeScript et JavaScript pour la gestion de la base de données.

Class Validator

Validation des données d'entrée avec des décorateurs.

HTML/CSS/JavaScript

Technologies front-end pour l'interface utilisateur.

API REST Endpoints

L'application expose les endpoints suivants :

GET /api/clients - Récupère tous les clients
GET /api/clients/:id - Récupère un client par son ID
POST /api/clients - Crée un nouveau client
PUT /api/clients/:id - Met à jour un client
DELETE /api/clients/:id - Supprime un client

Gestion des erreurs

L'application gère les erreurs suivantes :

  • 400 Bad Request : Données invalides (nom ou email manquant)
  • 404 Not Found : Client non trouvé
  • 500 Internal Server Error : Erreur serveur

Développement

Ce projet a été développé en respectant les meilleures pratiques de développement avec Nest.js :

  • Architecture modulaire et séparation des responsabilités
  • Validation des données avec des DTOs
  • Gestion des erreurs avec des exceptions personnalisées
  • Code typé avec TypeScript
  • Tests unitaires et d'intégration
